# TaskBoardPro

TaskBoardPro est une application Angular permettant de gérer des tâches avec une interface utilisateur simple et intuitive.

## Fonctionnalités

- **Navigation** : Accès aux différentes pages via une barre de navigation.
- **Compteur en temps réel** : Affichage du temps écoulé depuis le chargement de l'application.
- **Pages disponibles** :
  - Accueil
  - À propos
  - Tâches

## Prérequis

- Node.js (version 16 ou supérieure)
- npm (Node Package Manager)
- Angular CLI
